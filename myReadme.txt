Bouncing Ball Game

Description:
In order to win this game the user must not allow the ball to reach the bottom. It is something
like the pong game but just for one player. The user has to move the paddle to keep the ball bouncing.
A sound will be played everytime the ball touches the paddle. In order to win the game the user
must reach a minumum of 3 points, which will be added everytime the ball touched the paddle.
